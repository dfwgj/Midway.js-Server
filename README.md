# midway.js-Server

## QuickStart

see [midway docs][midway] for more detail.

### Development

```bash
$ yarn
$ yarn run dev
$ open http://localhost:7001/
```

### Deploy

```bash
$ npm start
```

### npm scripts

- Use `npm run lint` to check code style.
- Use `npm test` to run unit test.


[midway]: https://midwayjs.org
